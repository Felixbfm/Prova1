TCP/IP
Conjunto de protocolos que permite comunicação entre computadores na rede.
IP
Endereço numérico único de cada dispositivo na internet.
HTTP
Protocolo que transfere páginas e dados para o navegador.
HTTPS
Versão segura do HTTP com criptografia.
DNS
Traduz nomes de sites em endereços IP.
FTP
Protocolo para transferência de arquivos.
SMTP
Protocolo para envio de e-mails.
POP3 / IMAP
Protocolos para recebimento de e-mails.
VoIP
Transmissão de voz pela internet.
Cliente-Servidor
Cliente faz requisição e servidor responde.
Intranet
Rede interna de uma organização.
Extranet
Intranet com acesso externo controlado.
WWW
Sistema de páginas interligadas na internet.
Front-End
Parte visual com que o usuário interage.

Atributos

4° Parte - Formulários


Agora use < form action="O que fazer" method="post ou get" > e < /form > para criar formulários E < input type="TIPO" name="SEQUISER" > para as entradas
Tipos Famosos e como usar:
  1. < input type="text" name="nomedavariavel" >
  2. < input type="password" name="nomedavariavel" >
  3. < input type="radio" name="nomedavariavel" value="valor" > Mesmo nomes = Mesmo grupo
  4. < input type="checkbox" name="nomedavariavel" value="valor" > Mesmo nomes = Mesmo grupo
  5. < select name="O QUE QUISER" size="Tamanho que vai aparecer" > < option value="Valor" > texto < /option >< /select >
  6. < input name="quequiser" type="number" value="valorinicial" step="intervalos" min="valor" max="valor" >
  7. < input name="quequiser" type="range" value="valorinicial" step="intervalos" min="valor" max="valor" >
  8. < input type="color" name="quequiser" value="inicial" >
  9. < input type="date" name="quequiser" />
  10. < input type="email" >

E para enviar:
  1. < input type="submit" value="Nomezinho, Enviar" >
  2. < input type="reset" value="O que quiser" >
  3. < input type="button" value="?" >

Agora, 2 coisinhas especiais:
Pattern
< input type="tipo" name="nome" pattern="requisitos/forma" >
Antes de pattern adicione required se quiser obrigatório
Exemplo da maneira tem que ser, \ marca o caractere, pattern = "\d{x} - \d{y}
datalist
< input list="nome do datalist" name"id"
Ai em outro lugar você cria a datalist:
< datalist id="id"> < option value="valor" > < /datalist >
